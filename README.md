# IoT-Firstep
一个物联网(IoT)开发的入门教程。涉及单片机、上位机、移动应用、服务器后台开发的知识。以及蓝牙4.0、以太网模块的使用实例。

## 说明
本文的开发环境都是在Windows平台进行的。

> 说明：观察源码可以发现本repo没有4.x的部分。4.x是打算留给iOS开发或者小程序开发的，后续可能会添加，所以编号4.x暂时空出来。
